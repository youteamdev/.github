## Your Dev Team (YouTeam.dev)


> Your Own Dev Team (but run in github, with a self hosted dashboard)


## why you team

This project was my own attempt to build a fully automated software development project based entirely on GitHub ([self](https://github.com/bestony/self). However, I felt that [self](https://github.com/bestony/self) was an incomplete solution, so I am now continuing to further design and implement it, which led to You Team.
